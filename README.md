業務支援システム 操作マニュアル
1. 導入：ホーム画面への追加
迅速に起動できるよう、スマートフォンのホーム画面にアイコンを追加してください。

【iPhoneの方】

画面下部の「共有」ボタン（四角から上矢印が出ているマーク）を押します。
メニュー内の「ホーム画面に追加」を押し、右上の「追加」を選択します。
【Androidの方】

画面右上のメニュー（点が3つ縦に並んでいるマーク）を押します。
「ホーム画面に追加」を選択し、追加を押します。
2. 記録機能（乗務メモ）
地図上の、記録を残したい場所を押します（タップ）。
「行き先」を入力し、黄色の**「記録する」**ボタンを押すと保存されます。
→ 地図上に青い点が表示されます。
【修正・削除について】
地図上の青い点を押すと、**「編集」「削除」**ボタンが表示されます。行き先を書き直したい場合や、間違えて記録した場合はこちらから操作してください。

3. 履歴リストと分析
画面中央の**「履歴リスト」**ボタンを押すと、記録が曜日別に一覧表示されます。

地図での確認
リスト内の記録（カード）を押すと、地図がその場所へ移動します。「どのあたりでお乗せしたか」を地図上で振り返る際に便利です。
活用方法
高額利用のお客様や、注意が必要な人物の記録を蓄積することで、「どの曜日の、どの時間帯に多いか」をご自身で分析・確認できます。
操作
画面に入りきらない曜日は、リスト部分を横にスライドして確認してください。
【重要】情報管理について
このリストにはお客様の行動履歴などのプライバシー情報が含まれる可能性があります。
社外や第三者に画面を見せる行為は固く禁止します。

4. 時刻表機能
地図上の駅名を押すと、各路線の終電時刻が表示されます。
時刻変更時は「時刻編集」ボタンから数値を修正し、「完了」で保存してください。
補足
位置情報の利用許可を求められた際は、「許可」を選択してください。現在地が地図上に表示されます。
